Lógica de Programação e Algorítmos
com JavaScript

Uma introdução à programação de computadores
com exemplos e exercícios para iniciantes

Como o aprendizado de Algoritmos ocorre a partir do estudo das técnicas de programação e da prática de exercícios, este livro pretende ser uma importante fonte de conhecimentos para você ingressar nessa fascinante área da programação de computadores. Os capítulos cobrem os temas abordados na disciplina de Algoritmos: entrada e saída de dados, condições, repetições, vetores, strings, datas, funções com passagem e retorno de parâmetros, depuração de programas – além de tratar assuntos relacionados à construção de páginas Web com JavaScript.

Uma história intensa

Os tópicos discutidos em Lógica de Programação e Algoritmos são essenciais para quem quer mergulhar no mundo da Codificação de Computadores. Muitos iniciantes enfrentam desafios ao abordar essas matérias. Neste livro, o escritor aproveita seus mais de 15 anos de ensino em Algoritmos em cursos superiores para descomplicar o tema gradualmente. Cada seção foi meticulosamente estruturada para não sobrecarregar o leitor, sendo acompanhada por exemplos práticos e atividades para consolidar o aprendizado.

Autor dos Livros "O Ensino de Algoritmos apoiado por técnicas de Computação Afetiva" (Editora Novas Edições Acadêmicas, 2015), "Lógica de Programação e Algoritmos com JavaScript" (Editora Novatec, 2018) e "Lógica de Programação e Algoritmos com JavaScript - 2ª Edição" (Editora Novatec, 2022)

Nesta 2ª edição, além da atualização dos códigos e recursos do JavaScript, foram acrescentados três novos capítulos sobre Node.js, Express e React. Com Node.js é demonstrado como construir Algoritmos da forma “tradicional”, com a entrada e saída de dados via prompt de comandos. Com o Express é desenvolvida uma aplicação back-end com persistência em banco de dados – que recebe e envia dados no formato JSON. E, com o React, é criada a aplicação de front-end, que vai interagir com o usuário e acessar o back-end. São desenvolvidas rotinas de cadastro, pesquisa, exibição de dados estatísticos e criação de gráficos.

Estes e outros aspectos são explorados neste guia de programação que promete fazer você repensar o potencial do JavaScript no mundo do desenvolvimento!
